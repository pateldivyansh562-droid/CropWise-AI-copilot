# 📌 Overview

Kisan AI is an AI-powered agriculture platform designed to help Indian farmers with:

- 🤖 AI Farming Assistant
- 📈 Live Mandi Prices
- 🌱 Crop Disease Detection
- 🏛 Government Schemes
- 🌾 Land Management
- 📚 Farming Tutorials

The platform aims to make smart farming accessible, multilingual, and data-driven for every farmer across India.

---

# ✨ Features

## 🤖 AI Assistant
Ask farming-related questions in Hindi & English and get instant AI-powered responses.

### Supports:
- Crop guidance
- Fertilizer suggestions
- Pest management
- Irrigation advice
- Soil recommendations

---

## 📊 Live Mandi Prices
Track real-time mandi prices across different Indian markets.

### Features:
- Live crop rates
- Multiple mandi support
- Market trend updates
- Smart price tracking

---

## 🌱 Crop Doctor
AI-powered crop disease detection system.

### Features:
- Upload crop images
- Detect diseases instantly
- Treatment recommendations
- Prevention guidance

---

## 🏛 Government Schemes
Find and explore farmer-friendly government schemes.

### Includes:
- PM-Kisan
- Crop Insurance
- Irrigation Subsidies
- Farming Loans
- Agricultural Grants

---

## 🌾 Manage My Lands
Digital land & crop management system for farmers.

### Features:
- Track farming activities
- Maintain land records
- Crop history management
- Smart farming insights

---

## 📚 Farming Tutorials
Educational farming tutorials and smart agriculture techniques.

# 🚀 Getting Started

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/kisan-ai.git
cd kisan-ai
```

---

## 2️⃣ Install Dependencies

### Frontend

```bash
cd frontend
npm install
```

### Backend

```bash
cd backend
npm install
```

---

## 3️⃣ Setup Environment Variables

Create a `.env` file inside the backend folder:

```env
MONGO_URI=your_mongodb_uri
GEMINI_API_KEY=your_api_key
PORT=5000
```

---

## 4️⃣ Run the Project

### Frontend

```bash
npm run dev
```

### Backend

```bash
npm start
```
